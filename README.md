# SAE204-Cryptographie-collisions-de-md5-BELHARET-Adam-LACOSTE-Nathan

Répartition du travail : Adam sur le reverse MD5 et Nathan sur les collisions par préfixe. Recherche de bases de données de hachages (VirusTotal, NIST NSRL, MalwareBazaar). Étude de méthodes de collision par préfixe, mais les codes trouvés étaient inadaptés (SHA-256 au lieu de MD5). Début de la réalisation du diaporama de présentation. Téléchargement d'une base de données de 17,4 Go destinée au reverse MD5. Tentative d'exploitation du code trouvé lors de la séance précédente, mais celui-ci ne fonctionnait pas.
